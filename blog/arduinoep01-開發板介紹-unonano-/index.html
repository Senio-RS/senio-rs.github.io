ㄧ、Arduino UNO

這是無論接觸時間長短，最多人使用的Arduino開發板，也是最推薦初學者使用的開發板，因為網路上大部分的教學範例都是利用Arduino UNO進行介紹的，學習上比較沒有障礙。當然，我們接下來的文章也會以UNO進行解說。

Arduino Uno 是一款基於 ATmega328P 的微控制器板。常見的UNO版本有 14 個數位輸入/輸出接腳（其中 6 個可用作 PWM 輸出）、6 個類比輸入、USB連接孔（也可以作為供電孔，多數為Type B）、7~12V的DC電源插孔、1個16 MHz 陶瓷震盪器、ICSP 接頭和重置按鈕。因其開源製造的緣故，不同的副廠廠商會因產品定位做稍微的規格更改，因此可能在規格上會跟我們介紹的有些微出入，但通常不影響使用及日後的教學範例運作。

UNO的供電方式

  1. USB 連接埠： Arduino Uno 上有一個 USB Type-B 連接埠（可能因為版本而有不同），通常用來將 Arduino Uno 連接到電腦或其他 USB 電源供應器上。透過 USB 連接埠供電時，Arduino Uno 的電源會來自連接的 USB 介面，一般情況下，這提供了 5V 的電源，也是最常見的供電方式。另外，這個連接埠也常作為程式上傳、維護使用。因此，倘若讀者只是作為學習使用，將UNO連接在電腦上即可，無須煩惱其供電問題。

  2. 電源輸入插座： Arduino Uno 的邊緣有一個圓形的DC插座，通常用來連接外部電源供應器。這個插座接受 7V 到 12V 的直流電源，若供電電壓低於 7V，則需要注意 5V 腳位的輸出電壓可能不足。透過這個插座供電時，電源進入 Arduino Uno 的 DC 插座，並由板上的穩壓器轉換成 5V 電源，以供板上的元件使用。這是在需要獨立供電，或者需要更高電壓的情況下常用的方式。

  3. Vin 腳位： Arduino Uno 上有一個標有 “Vin” 的腳位，它代表 “Voltage In”。這個腳位可以用來直接輸入電壓到 Arduino Uno 的電源系統。這個腳位接受的電壓範圍與電源插座相同，即 7V 到 12V 的直流電壓。同樣地，這個電壓會經過板上的穩壓器轉換成 5V 電源。這個腳位的使用情境通常與電源插座類似，但是更多地被用於自定義電源連接。 其他有關供電的腳位，請參考以下表格。

Arduino UNO的各種腳位

Arduino開發板的腳位主要是作為供電、連接模組使用，Arduino UNO共提供數位輸入/輸出、類比輸入、電源供應三種類型的腳位。

Note

類比訊號和數位訊號的差別在哪裡？

類比訊號是連續變化的信號，其值在一定範圍內不斷變化，就像是一條曲線。而數位訊號是離散的，只能取有限個值，如0和1，就像是由一系列獨立的點組成的。

數位輸入、輸出腳位

Arduino的開發板一般來講會提供14個數位輸出/輸入腳位，分別是D0到D13。其中，D3、D5、D6、D9、D10、D11這六隻腳位可以作為PWM使用；D10、D11、D12、D13可以搭配 SPI Library 可提供 SPI 序列通訊。

另外，一般來講，開發板會預設一個由D13控制的LED閃爍程式，控制D13旁邊標示為L的LED閃爍。因為通電就會自動運行，因此可以藉此來檢視開發板是否正常。

Note

PWM(Pulse Width Modulation)

這裡稍微提及一下什麼是PWM(Pulse Width Modulation)，簡單來說PWM是一種模擬類比輸出的技術，透過過改變信號在開啟（高電位）和關閉（低電位）之間的時間比例來模擬在電路板的5V和0V之間的電壓（類似於類比訊號）。在頻率高的情況下得以模擬出與類比訊號相似的訊號，一般來說， Arduino 的 PWM 頻率約為 500Hz，也就是訊號間隔約2ms。

類比輸入腳位

除了數位腳位，Arduino還提供了A0-A5共6支類比「輸入」腳位。Arduino的類比電壓讀取是10bit，也就是將電壓讀取分為2^10 = 1024等分。舉例來說，當操作電壓是5v時，一等分的電壓就是5v/1024=0.0048v。另外，值得注意的是A0~A5腳位並不能作為類比輸入腳位使用。

以下是UNO的類比腳位的詳細功能：

  • A0 - 類比輸入腳位：這個腳位可以讀取0到5伏特之間的類比電壓值，對應到Arduino的0到1023的數位值。
  • A1 - 類比輸入腳位：與A0相似，可以讀取類比電壓值。
  • A2 - 類比輸入腳位：同樣可以讀取類比電壓值。
  • A3 - 類比輸入腳位：和前面的類比輸入腳位相同，用於讀取類比電壓。
  • A4 - 類比輸入、數位輸出腳位：除了可以作為類比輸入腳位外，還可以作為I2C通信的SDA數位腳位。
  • A5 - 類比輸入、數位輸出腳位：類似於A4，除了類比輸入外，還可用於I2C通信的SCL數位腳位。

Tip

在實務操作時，A0~A5雖然是作為類比訊號輸入使用，但是其實他也可以作為數位輸入、輸出訊號使用。當D0到D13不夠用的時候，可以作為臨時使用。

供電腳位

  • 5V腳位：Arduino Uno板上有一個5V接腳，可以提供5伏特的直流電源。這個接腳是由板上的USB插口供電或者通過外部直流電源供電，可以提供外部偵測模組供電。
  • 3.3V腳位：除了5V接腳外，Arduino Uno還有一個3.3V接腳，提供3.3伏特的直流電源。這個接腳可以用來給一些低功耗的外部模塊或者感測器供電。
  • Vin腳位：Arduino Uno板上有一個Vin（Voltage In）接腳，可以接受外部直流電源供應。這個接腳通常用於接駁外部電源模塊或者電池。
  • GND腳位：這是UNO提供的兩個接地的腳位，通常會與5V、3.3V腳位共同使用。

重置鈕及Reset腳位

按下USB旁邊的重置鈕可以使開發板重新執行使用者寫入的程式。另外，當你使Reset 腳位為低電位時，也會產生相似的效果。

二、Arduino Nano

Nano的供電方式

  1. USB 供電：Arduino Nano 可以通過 USB 線連接到電腦或 USB 電源插座。當通過 USB 供電時，Arduino Nano 的板子會從 USB 接口獲取電源，同時也可以通過 USB 連接埠與電腦通信。

  2. Vin腳位供電：除了 USB 供電外，Arduino Nano 也可以通過外部直流電源供電。在 Arduino Nano 的板子上有一個 Vin 腳位，你可以通過這個腳位接入外部電源，例如 7V 至 12V 的直流電源。這樣做時，注意極性和電壓的適配。

Info

注意！

Arduino Nano和 UNO最大的差別在於體積。為了輕便及成本考量，在Nano上面並沒有提供DC直流插座。因此，在製作專案時，除了自焊USB插座，可以多多利用Vin腳位進行供電。
